## 스터디 방식
- 매일 30분씩, 한 챕터씩 공부해오고 정리본 노션에 공유하기

- 스터디 발표자는 그날그날 자유롭게 정하기
- 서로의 실력 향상이 목표이기 때문에 별도의 벌금은 없지만, 책을 못 읽었더라도 최대한 할 수 있는 만큼 해오기!
